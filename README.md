## 京东移动端首页

#### 一：项目描述

​	1、京东移动端首页是基于百分比布局的来适应移动端不同设备的学习式项目

​	2、运用到了经典布局“双飞翼布局”，“两栏自适应”等布局，

​	3、还运用到了手机点击事件，轮播图的手势滑动，iscroll滑动插件等
